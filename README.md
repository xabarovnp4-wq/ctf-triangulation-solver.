# CTF Triangulation Solver

Automated solver for the triangulation CTF challenge.

## Usage

1. Open in Gitpod: [![Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/YOUR_USERNAME/ctf-triangulation-solver)
2. The solver will run automatically
3. Wait for the flag!

## Challenge
Solve 120 rounds of triangulation problems to get the flag.
